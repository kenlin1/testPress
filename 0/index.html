# 什么是Java

Java是一种面向对象的编程语言，它是由Sun Microsystems公司于1995年发布的，现在由Oracle公司拥有和维护。它是一种跨平台的语言，可以在多种计算机平台上运行，包括Windows、Mac OS和Linux等操作系统。

Java是一种面向对象的编程语言，它的基本概念是“对象”，它的基本结构是“类”。Java程序是由一系列类组成的，每个类都有自己的属性和方法，它们之间可以相互调用，从而实现程序的功能。

Java的特点是它是一种解释型语言，也就是说，它的程序不需要编译，只需要将源代码转换成字节码，然后由Java虚拟机（JVM）来执行字节码，从而实现程序的运行。

Java还具有许多优点，比如安全性、可移植性、可扩展性、高性能等。它的安全性体现在它可以防止程序员犯错，防止恶意程序的入侵；可移植性体现在它可以在多种计算机平台上运行；可扩展性体现在它可以通过扩展库来支持新的功能；高性能体现在它可以提高程序的运行效率。

总之，Java是一种面向对象的编程语言，它具有安全性、可移植性、可扩展性和高性能等优点，是当今最流行的编程语言之一。
